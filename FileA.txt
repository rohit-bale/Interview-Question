1. Web scraping

You’ll find no shortage of excellent (and free) public data sets on the internet, and you might want to show prospective employers 
that you can find and scrape your own data as well. Plus, knowing how to scrape web data means you can find and use data sets that
 match your interests, regardless of whether or not someone has already compiled them.
If you know some Python, you can use tools like Beautiful Soup or Scrapy to crawl the web for interesting data. If you don’t know how
 to code, don’t worry. You’ll also find several tools that automate the process (many offer a free trial), like Octoparse or ParseHub.
